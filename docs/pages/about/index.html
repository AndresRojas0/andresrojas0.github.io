Acerca del sitio

Sitio web pensado para compartir casos de uso cotidiano con Inteligencia Artificial generativa y herramientas de LLM, desde la perspectiva de una persona que trabaja en rubros no directamente relacionados, experimentando de forma autodidacta.

La misión de este blog es recopilar los mejores casos de uso de desarrollo con IA, documentando el proceso (errores, aciertos y prompts concisos) para optimizar la interacción con las herramientas.

La visión en un futuro es ofrecer soluciones con aplicaciones web mobile first a potenciales negocios, clientes y amigos, a partir de la estructura de datos definida por IA.

Los valores y principios se centran en la creencia de que existe una oportunidad real de mejora en procesos manuales mediante la IA, buscando siempre la creación de un Producto Mínimo Viable (MVP) funcional para un negocio, fomentando la creatividad y el diálogo sobre su posterior desarrollo e implementación.
