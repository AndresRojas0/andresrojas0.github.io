Acerca del sitio

Sitio web pensado para compartir casos de uso cotidiano con herramientas de IA desde la perspectiva de una persona que trabaja en rubros no directamente relacionados, experimentando de forma autodidacta.

La misión de este blog es recopilar los mejores casos de uso de desarrollo con IA, errores y aciertos.

La visión en un futuro es ofrecer soluciones con aplicaciones web a potenciales negocios, clientes y amigos.

Los valores y principios se centran en mostrar que existe una oportunidad de mejora, pensando en un producto mínimo viable para un negocio. A partir de ahí, las oportunidades para conversar acerca de crear y desarrollar la o las soluciones con la implementación que corresponda.
